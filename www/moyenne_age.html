Âges auxquels sont battus les records

On souhaite sortir directement la liste pour chaque athlète [record_athlète, nom, âge_lors_du_record]. Cette liste permettrait alors de déduire les âges, en moyenne, auxquels sont battus les records pour les épreuves 800, 1500, 3000, etc. Malheureusement on ne trouve pas le moyen de faire cela en une seule requête SQL :-(
A défaut essayons de procéder progressivement. On peut commencer sortir le record et le nom. P.ex. pour le 3000 :

SELECT MAX(points) AS t_points, nom AS t_nom
FROM ligne
WHERE club =  'Ac paris-joinville'
AND idEpreuve = 3000
GROUP BY nom;  

(à noter que le GROUP BY permet d'éviter que
 MAX isolément ne sortirait qu'une seule ligne)

Résultat :
 
752	ABDELKADER
776	ANDRIEU
697	BALAGE
872	BARAMI
...
On peut alors itérer sur le tableau du résultat. C'est possible mais c'est coûteux de faire une requête pour chaque ligne. Il serait nettement préférable de tout faire en une seule requête, mais comment ? Il faudrait l'usage d'une variable mais en SQL standard est-ce possible ? 

-- itération sur le tableau [t_points, t_nom] :
  SELECT nom, (annee - naissance) AS age
  FROM ligne
  WHERE points = [t_points]
    AND nom = [t_nom]
    AND idEpreuve = 3000;
    
 Cela donnerait :
 
752	ABDELKADER 37
776	ANDRIEU    35
697	BALAGE     30
872	BARAMI     30
...

1/ que faire si un athlète bat son record à 25 puis l'égale exactement à 31 ans, retenir lequel de ces deux âges pour le calcul de la moyenne indiquant l'âge auquel est battu le record ?
