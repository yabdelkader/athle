Nombre de participations de chaque athlète

... en comptant une par année

SELECT nom, count(DISTINCT(annee)) AS participations 
FROM IC
GROUP BY nom
ORDER BY participations DESC

Ce classement ne figurait initialement pas dans la page. C'est à la demande de Juliette B. (2 participations).


... avec plusieurs épreuves par tour

SELECT nom, count(*) AS participations, GROUP_CONCAT( DISTINCT epreuve SEPARATOR ',' ) 
FROM IC
GROUP BY nom
ORDER BY participations DESC

SELECT nom, SUM(points), count(id), AVG(points) as total FROM IC

Avec meilleure épreuve par tour

A la demande de Thibault : « qui a rapporté le plus de points en cumulé avec sa meilleure épreuve par tour ? »

Ce critère est juste pour les athlètes qui participent à plusieurs épreuves ou font une contre perf. De plus calcul par année et pas par tour, car c'est plus simple et ça évite de pénaliser les athlètes avec accidents (chute à un 800 à Créteil par exemple).

Alors allons-y. Commençons par un GROUP BY nom, annee pour retenir le MAX, puis encapsulement dans un table temporaire (on peut faire plus élégant mais là je vois pas. Si un(e) fort(e) a une suggestion...)

SELECT n, AVG(pts) AS moy
FROM (
  SELECT annee AS a, nom AS n, max(points) AS pts
  FROM IC
  GROUP BY nom, annee
) tmp
GROUP BY n
ORDER by moy DESC
