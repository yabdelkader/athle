Nombre de participations de chaque athlète

... en comptant une par année

SELECT nom, count(DISTINCT(annee)) AS participations 
FROM IC
GROUP BY nom
ORDER BY participations DESC

Ce classement ne figurait initialement pas dans la page. C'est à la demande de Juliette B. (2 participations).


... avec plusieurs épreuves par tour

SELECT nom, count(*) AS participations, GROUP_CONCAT( DISTINCT epreuve SEPARATOR ',' ) 
FROM IC
GROUP BY nom
ORDER BY participations DESC

Plus de points...

... par épreuve

SELECT epreuve, count(id), SUM(points) as total FROM IC
GROUP BY epreuve
ORDER BY total DESC
