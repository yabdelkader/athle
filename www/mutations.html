Méthode pour trouver les athlètes ayant appartenu à 'n' clubs

On pourrait naïvement itérer sur chaque athlète un par un. Mais il doit être possible de procéder de façon meilleure.

Par exemple on pourrait chercher les athlètes ayant au moins 3 clubs différents ainsi : 

SELECT DISTINCT  l1.nom, l1.prenom, l1.club, l2.club, l3.club
FROM ligne l1, ligne l2, ligne l3
WHERE l1.nom = l2.nom AND l1.prenom = l2.prenom AND l1.club != l2.club 
  AND l1.nom = l3.nom AND l1.prenom = l3.prenom AND l1.club != l3.club 
  AND l2.club != l3.club

static void readCSV() {

	try (BufferedReader br = new BufferedReader(new FileReader("mutations.csv"))) {
		List<Person> persons = br.lines()
		.map(mapToPerson)
		.collect(Collectors.toList());
	} catch (IOException e) {
		e.printStackTrace();
	}
}

// fonction lambada pour créer une personne
public static Function<String, Person> mapToPerson = (line) -> {
	String[] p = line.split(", ");
	return new Person(p[0], p[1], p[2], p[3], p[4]);
};
